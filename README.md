#### 为什么项目名是 `hexo_s`？

1. 因为博客最早是使用 [Hexo](https://github.com/hexojs) 搭建的

2. 中途手机炸了一次，补救后备份到 GitHub 上，不假思索的取的项目名

#### 在本地查看网页

切换到工作目录并运行 `pnpm i && pnpm build && pnpm preview`

**不要使用 `pnpm dev`，因为有未知问题会导致跑不起来**
